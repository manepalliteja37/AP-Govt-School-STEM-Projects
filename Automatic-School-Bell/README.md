# Automatic School Bell System

Designed by MBT  
Telugu Mad Thinker  

## Features
- 8 Period System
- 5 Minutes per period(customised)
- 10 Minutes Lunch Break(customized)
- OLED Countdown Display
- Buzzer Notification
- EEPROM for configuration storage

## Hardware
- Arduino UNO
- OLED Display
- RTC Module
- Buzzer
- LED
