# 🥭 Ugadi Pachadi Electronics Project

Ugadi pachadi is not just food…
It represents a **blend of emotions ❤️**

This project connects those emotions with **electronics concepts ⚡**

---

## 🎯 Concept

Each taste in Ugadi pachadi → mapped to:

* Emotion 😄😖🔥
* Electronics concept ⚡

Example:

* Sweet → Love → Stable Voltage
* Bitter → Failures → Noise Signal
* Spicy → Anger → Overcurrent
* Salty → Tears → Power Loss

---

## 🧰 Components Used

* Arduino Nano
* ST7735 TFT Display
* LEDs (5 colors)
* Push Button
* Resistors (220Ω)

---

## 🔌 Wiring

### TFT Display (SPI)

| TFT | Arduino |
| --- | ------- |
| VCC | 5V      |
| GND | GND     |
| SCK | D13     |
| SDA | D11     |
| CS  | D10     |
| DC  | D9      |
| RST | D8      |

### Other Components

* LEDs → D2 to D6
* Button → D7
